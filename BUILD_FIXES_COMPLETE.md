## Technical Notes

### Font API Decision
The standard `Font(float height, int styleFlags)` constructor is used throughout JUCE and is the correct approach for styling fonts. The `FontOptions` API exists for advanced font configuration but is not needed for simple styling.

See [FONTOPTIONS_ERROR_ANALYSIS.md](FONTOPTIONS_ERROR_ANALYSIS.md) for detailed analysis.

### Why Projucer Regeneration Is Required
AU plugins require specific build phases, Info.plist configurations, and code signing settings that are generated by Projucer from the .jucer file. These cannot be manually created or fixed through code changes alone.

See [AU_BUILD_FIX.md](AU_BUILD_FIX.md) for complete explanation.
