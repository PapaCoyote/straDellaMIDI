# straDellaMIDI

A MIDI instrument plugin (AU / VST3) built with [JUCE](https://juce.com/) and the Projucer.

## Getting Started

1. Install [JUCE](https://juce.com/get-juce) and the Projucer.
2. Open `straDellaMIDI.jucer` in the Projucer.
3. Set the path to your JUCE installation in the Projucer's Global Paths settings.
4. Export the project for your target IDE (Xcode for AU/VST3 on macOS, Visual Studio for VST3 on Windows).
5. Build the plugin from the exported IDE project.

## Plugin Formats

- **AU** (Audio Unit) — macOS only
- **VST3** — macOS, Windows, Linux
